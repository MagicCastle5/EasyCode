EasyCode: Beginner's Guide

Syntax Basics

Comments

* This is a comment

Variables

Make x = 5 Make name = 'Alice' Make total = x + 10 Make len = Long(name)

Output

Show -Hello world!- Show x

Logic

If Statement

When x > 3 Show -x is bigger than 3-

While Loop

Make i = 0 While i < 5 Show i Make i = i + 1

Functions

Define a Function

Function Greet(name) Show -Hello - Show name End

Call a Function

Start Greet('Alice')

Expressions & Logic

  • <> means not equal (!=)
  • & means and
  • | means or
  • ~ means not
  • Long(x) gives length of x

Sample Program

Make name = 'Bob' Function Welcome(n) Show -Welcome - Show n End Start Welcome(name) Make i = 0 While i < 3 Show -Looping...- Make i = i + 1

Tips

  • Indent blocks with 4 spaces
  • Use -text- for string display
  • Variable names are case-sensitive
  • Use only defined variables to avoid errors
